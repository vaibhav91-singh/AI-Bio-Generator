## 🚀 Quick Start (3 Steps)

### Step 1: Setup
```bash
# Windows
setup.bat

# Mac/Linux
chmod +x setup.sh && ./setup.sh
```

### Step 2: Start
```bash
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend
cd frontend && npm start
```

### Step 3: Open
Visit `http://localhost:3000` - That's it!

## 🆘 Troubleshooting

### "API not connecting"
→ Check backend is running
→ Check REACT_APP_API_URL is correct
→ Check CORS_ORIGIN matches

### "Ads not showing"
→ AdSense needs approval (5-7 days)
→ Verify publisher ID is correct
→ Check domain matches AdSense

### "No bios generating"
→ Check HuggingFace API key
→ Try without API key (uses fallback)
→ Check keywords are valid

### "Slow page"
→ Run Lighthouse audit
→ Check image sizes
→ Verify API response times

For more: See README.md troubleshooting section
